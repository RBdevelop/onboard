== Onboard  

A simple onboarding app. Designed for training remote employees and sharing content in one hub.


Current employing Dashing-Rails. A Dashing IO library for rails backend. Please install Dashing Rails dependings to run 

Gems: Redis, JQuery 3.0.4, Puma, and Dashing Rails. 

Database: PostgreSQL 



Still in development. To be customized with Organization specific information.
